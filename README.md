# User Discovery App

A beautiful React application that fetches and displays random user profiles from an API, showcasing modern UI/UX design patterns.

## Features

- Fetches random user data from FreeAPI
- Beautiful card-based grid layout
- Profile cards with avatars, contact info, and location
- Responsive design (mobile & desktop)
- Smooth hover animations
- Loading and error states

## Tech Stack

- **React** - UI framework
- **Vite** - Build tool
- **CSS** - Custom styling with modern features

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
